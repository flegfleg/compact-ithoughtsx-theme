# Compact Theme for iThoughtsX

Theme for [iThoughtsX](https://www.toketaware.com/ithoughts-osx) in 4 flavours: 

* Dark
* Light
* Mono Dark
* Mono Light

Install the [SF Compact font](https://developer.apple.com/fonts/) for best results.
